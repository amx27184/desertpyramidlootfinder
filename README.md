RECCOMENDED CHANGING DEFAULT REQUIREMENTS!!

-------------------------------------------

              DEFAULT:

            VERSION 1.17
            
CHANGE IT BY CHANGING THE VERSION VALUE AT START OF THE CODE
(MCVersion version = MCVersion.v1_17)
CHANGE .v1_17 WITH VERSION DESIRED (REMEMBER TO USE UNDERSCORES INSTEAD OF THE PERIOD)
EXAMPLE = .v1_15
DO NOT USE VERSIONS PRIOR TO VERSION 1.3 (DESERT PYRAMIDS WERE ADDED IN 1.3, SO SETTING A VERSION PRIOR TO VERSION 1.3 WILL MAKE THE CODE CRASH)
